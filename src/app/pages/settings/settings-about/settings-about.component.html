Sign Language Translation Platform

What We Do

This is a modern Angular-based web application designed to bridge communication gaps between sign language and spoken languages. The platform enables users to translate text and audio input into sign language visuals (using FSW notation and pose rendering), supporting multiple spoken languages and signed languages globally.

Built with Ionic Framework for cross-platform compatibility and NGXS for state management, the application prioritizes accessibility, user experience, and seamless translation workflows.

Our Mission

The application facilitates bidirectional translation between spoken/written languages and sign language, making communication more inclusive for deaf and hard-of-hearing communities. Users can input text or speech, select source and target languages, and receive visual sign language output rendered through specialized pose viewer components.
